# Vue 7-Day Mastery

This repository is a mono repo documenting a structured 7-day learning program focused on mastering Vue 3 through incremental, hands-on projects. Each day builds on the previous one, with an emphasis on architectural understanding, correct mental models, and practical application.

---

## Learning Progress

### Day 1: Todo App, Vue Fundamentals  
https://github.com/imanatsaqif/vue-7-day-mastery/tree/main/todo-app

Focus:
- Vue 3 fundamentals using the Composition API
- Reactive state with `ref` and `reactive`
- Template syntax and directives
- Basic event handling and conditional rendering

---

### Day 2: Todo App, Component Architecture  
https://github.com/imanatsaqif/vue-7-day-mastery/tree/main/todo-app

Focus:
- Component decomposition
- Props and emits for parent-child communication
- Slot usage for flexible component design
- Improving code structure and reusability

---

### Day 3: Portfolio Website, Routing Fundamentals  
https://github.com/imanatsaqif/vue-7-day-mastery/tree/main/portfolio

Focus:
- Vue Router fundamentals
- Dynamic routes and route parameters
- Layout and view separation
- Route guards for data validation
- Static data management

---

### Day 4: Portfolio Website, State Management with Pinia  
https://github.com/imanatsaqif/vue-7-day-mastery/tree/main/portfolio

Focus:
- Centralized state management using Pinia
- Store-driven data access and business logic
- Theme management with persistence
- Refactoring components to consume store state
- Integrating stores with router guards

---
